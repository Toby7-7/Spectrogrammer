# Fork 改动说明

本文记录当前仓库相对上游 `Spectrogrammer` 的主要改动，重点覆盖 Android 构建、运行时行为、双声道频谱显示、瀑布图、配置系统和中英文界面。这里描述的是“当前分支已经做成了什么”，不是临时提交草稿。

## 项目定位变化
- 开发重心从通用实验性频谱查看器，转到 Android 手机竖屏使用场景。
- UI 默认语言改为英文，并支持在应用内切换到简体中文。
- 处理链调整以“稳定显示真实频谱”为先，避免为了试验性优化牺牲基础可靠性。

## 1. 构建与仓库整理
- `Makefile`
  - 重做 Android 构建和打包流程，补齐 `init-submodules`、`doctor-android`、`push`、`run`、`logcat`、`clean`
  - 自动探测 Android SDK、NDK 和 build-tools
  - 编译 `src/java` 里的前台服务并打入 APK
  - 默认 APK 名仍为 `Spectrogrammer.apk`
- `README.md`
  - 改成面向当前分支的英文主说明
- `README.zh-CN.md`
  - 新增中文镜像文档
- `docs/android-build-install.md`
  - 用英文记录当前环境下 Android 构建、mDNS 发现和 Wi-Fi ADB 安装流程
- `docs/fork-differences.md`

## 3. 配置持久化与默认值迁移
- `src/app/AppConfig.h`
- `src/app/AppConfig.cpp`
  - 新增版本化配置文件与迁移逻辑
  - 当前配置版本已提升到 `11`
  - 新增和持久化的关键配置包括：
    - 界面语言
    - 声道模式
    - 交换左右顺序
    - 显示上方频谱图
    - 图上标注字号
    - 图上标注透明度
    - 输入增益
  - 统一默认值：
    - 语言 `English`
    - 音频源 `默认`
    - 声道模式 `双声道独立`
    - 采样率 `自动 / 48000 Hz`
    - 输入增益 `0 dB`
    - FFT `4096`
    - 平滑因子 `0.10`
    - 峰值标记来源 `短时峰值`
    - 峰值保持开启，默认 `4 秒` 回落

  - 对配置范围做兜底：
    - `滚动速度` 下限 `2 ms`
    - `FFT` 上限 `8192`
    - `峰值回落时长` 范围 `0` 到 `120 s`
    - `输入增益` 范围 `-24 dB` 到 `+24 dB`
    - `0` 表示 `不回落`

## 5. 主界面与交互重做
- `src/app/Spectrogrammer.cpp`
- `src/app/Spectrogrammer.h`
  - 主界面和设置支持中英文切换
  - 顶部工具栏当前为四个按钮：`暂停/继续`、`清除峰值`、`清除游标`、`设置`
  - 增加状态行，显示输入采样率、当前视图范围、FFT 大小和频率分辨率
  - 设置整理为分类页面，并增大字体、控件和边距，适配手机触控
- 主图交互
  - 单指点击或拖动，可移动手动游标
  - 游标会在频谱图和瀑布图上同时显示纵线，并显示当前频率与 dB
  - 顶部可一键清除手动游标
  - 双指左右可缩放和平移频率视图，便于观察局部频段
- 双声道显示
  - 支持 `单声道`、`左声道`、`右声道`、`双声道混合`、`双声道相减`、`双声道独立`
  - 双声道独立模式支持 `交换左右顺序`
  - 当前目标设备上的物理左右麦克风映射已经校正
- 峰值相关
  - 增加峰值保持曲线
  - 峰值标记可在 `实时峰值` 和 `短时峰值` 间切换
  - 峰值回落时长支持可调，也支持 `不回落`
- 图上文字
  - 支持调节图上标注字号与透明度
  - 双声道独立时使用简化的 `左 / 右` 或 `L / R` 标签

## 7. 当前可调功能总览
- 音频源：`默认 / 通用 / 语音识别 / 摄像机 / 未处理`
- 声道模式：`单声道 / 左声道 / 右声道 / 双声道混合 / 双声道相减 / 双声道独立`
- 输入增益：`-24 dB` 到 `+24 dB`
- 采样率：`自动` 或固定档位，实际最高可用取决于设备支持
- FFT 大小：`128` 到 `8192`
- 抽取级数：`0` 到 `5`
- 窗函数：`矩形窗 / 汉宁窗 / 汉明窗 / 布莱克曼-哈里斯窗`
- 滚动速度：`2 ms` 到 `250 ms`
- 指数平滑因子：`0.00` 到 `0.95`
- 图上标注字号：`0.7` 到 `1.8`
- 图上标注透明度：`0.25` 到 `1.0`
- 频率轴刻度：`线性 / 普通对数 / 音乐对数 / Mel / Bark / ERB`
- 频谱/瀑布可见性：`显示上方频谱图 / 显示瀑布图`
- 瀑布图高度：`1/4 屏 / 1/3 屏 / 2/5 屏 / 1/2 屏 / 3/5 屏 / 2/3 屏 / 3/4 屏`
- 峰值标记数量：`0 / 1 / 3 / 5`
- 峰值标记来源：`实时峰值 / 短时峰值`
- 峰值回落时长：`0` 到 `120 s`
- 运行控制：`后台采集 / 保持亮屏`

## 8. 当前状态与注意事项
- 当前仓库以 Android 手机使用体验为优先目标，Linux 构建入口仍保留。
- 高采样率档位和 `未处理` 音频源是否可用，仍依赖具体设备驱动。
- 固定高采样率如果设备不支持，程序会尽量回退到更低的可用档位。
- 这份说明聚焦源码和功能改动，不包含 APK、dex 或工具工作区临时文件等本地临时产物。
